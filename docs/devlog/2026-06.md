## 2026-06-23

### Summary

Added deterministic clip-level LFO support to the sequencing engine.

### Decisions

- LFOs are beat-clocked and stateless: values derive from beat, rate, phase offset, shape, amplitude, and offset.
- MVP LFO destinations are limited to MIDI CC values and probability modulation.
- LFO modulation is evaluated during engine rendering before CC and note events are emitted.

## 2026-06-23

### Summary

Added beat-range windowed rendering to the sequencer engine.

### Decisions

- Whole-loop rendering now delegates to the windowed render path.
- Windowed rendering consumes probability deterministically from beat zero so a window matches a full render filtered to that same beat range.
- Events are emitted for `[startBeat, endBeat)`, including note-offs inside the window for notes started earlier.
